FAQ

How to select my own Dictionary file?

Click the analysis button to start your analysis, you’ll be asked to either select the existing dictionary file or to drop your own dictionary file. Once you select the button to drop your own dictionary file, you will be asked to upload your dictionary file from your local machine.

What is a Dictionary file?

A dictionary file contains domains and subdomains with respect to the fields such as hospitals, student etc. With the help of this file, analysis will be done.

Do we get to select our own comment file?

Yes, you can select your own comment file

Will be able to see all the domains before I select them for analysis?

Yes, after dropping your dictionary file, you will be given two options. Either to select all the existing domains, or to show all the domains. You can click on “no” option to display all domains.

How is the result generated?

The result is generated in a form of bar graph, where purple shows positive feedback and lavender shows negative feedback. You can hover your mouse on the bars, to see the respective field in line graph.

If I want to analyse the data for only two domains will I be able to do it?

After the analysis, you will find an option to apply the filter to select domains for which you want to carry the analysis.

Is there an option to download the output? If yes, how to download it?

In the results page, you will find an option to download output, once you click on that, the output will be automatically downloaded in a form of an excel sheet.

Whom should I contact for general enquiries?

There is a contact us section in the home page, you can either email us or contact us through the number given.
